# Project Overview — react-ui-kit

Résumé
- Nom : react-ui-kit (monorepo)
- Objectif : bibliothèque de composants UI réutilisables pour applications React, documentée avec Storybook, testée et packagée.
- Scope : composants accessibles, design system léger, exemples d'utilisation, et guide pour contributors.

Principaux objectifs business / techniques
- Fournir des composants réutilisables, testés et stylés avec TailwindCSS.
- Faciliter l’intégration dans des apps React (TypeScript).
- Disposer d’une documentation Storybook complète et d’une CI solide.
- Servir de base pour démontrer compétences frontend dans le portfolio.

Livrables attendus
- packages/ui : composants, build, types
- packages/storybook : documentation interactive
- CI : lint/test/build/storybook build
- README + guides d’architecture + UI/UX guidelines

Utilisateurs cibles
- Développeurs front-end
- Designers souhaitant voir les composants en action
- Recruteurs / examinateurs techniques
